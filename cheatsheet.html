Git & Bash Command Cheat Sheet

A quick reference guide for essential Git and Bash commands.

Basic Git Commands
  • git init - Initialize a new Git repository
  • git clone <repo-url> - Clone a repository
  • git status - Check repository status
  • git add <file> - Stage a file
  • git commit -m "message" - Commit changes
  • git push - Push changes to remote repository
  • git pull - Fetch and merge changes from remote
Branching and Merging
  • git branch - List all branches
  • git checkout <branch> - Switch branch
  • git merge <branch> - Merge a branch
  • git branch -d <branch> - Delete a branch
Remote Repositories
  • git remote -v - Show remote repositories
  • git remote add origin <url> - Add a remote repository
  • git push -u origin <branch> - Push changes
Undoing Changes
  • git reset <file> - Unstage a file
  • git reset --hard <commit> - Reset to a commit
  • git revert <commit> - Revert a commit
Useful Git Commands
  • git log - View commit history
  • git diff - Show differences
  • git stash - Save changes temporarily
  • git stash pop - Apply and remove last stash
Basic Bash Commands
  • pwd - Show current directory
  • ls - List files in the current directory
  • cd <directory> - Change directory
  • mkdir <dir-name> - Create a new directory
  • touch <file-name> - Create an empty file
  • rm <file> - Remove a file
  • rm -r <dir> - Remove a directory
  • cp <source> <destination> - Copy files or directories
  • mv <source> <destination> - Move or rename files
